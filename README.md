# 🖼️ Image Editing Web Application

An interactive **Image Editing Web Application** built using **Python** and **Streamlit**. This app allows users to upload or capture images and perform a wide range of image processing operations in real-time through an intuitive and user-friendly interface.

---

## 🚀 Features

### 📥 Image Input
- Upload images (`JPG`, `PNG`)
- Capture images using webcam

---

### 🛠️ Basic Editing
- Convert to Grayscale  
- Color Inversion  

---

### 🔄 Transformations
- Resize image (custom width & height)  
- Rotate image  
- Flip image (Horizontal & Vertical)  

---

### 🎛️ Adjustments
- Brightness control  
- Contrast control  

---

### 🎨 Filters & Effects
- Blur  
- Sharpen  
- Edge Detection  


---

### ✏️ Annotation Tools
- Add custom text  
 

---

### 🔍 Comparison Tool
- Interactive **Before/After slider** to compare original and edited images  

---

### 📤 Output
- Real-time preview of edits  
- Download edited image  

---


## 🌐 Deployment

The application was successfully deployed online using **Streamlit Cloud**, allowing users to access the app through a web browser without installing any software.

### 🔧 Deployment Process

The deployment process included:

- Uploading the project files to GitHub  
- Connecting the GitHub repository to Streamlit Cloud  
- Configuring the main application file (`app.py`)  
- Automatically building and hosting the app online
